# Session: Add research backlog item — organisational intent formal specification

**Date:** 2026-03-14
**Slug:** add-organisational-intent-formal-spec

## What was done

Created `Research/backlog/2026-03-14-organisational-intent-formal-specification.md` from the issue request.

The research question asks whether organisational intent — mission, values, strategy, resource allocation — can be expressed as a formally structured specification from which human-readable artefacts are derived, and against which OKRs, funding decisions, and initiative prioritisation can be continuously checked for logical consistency and derivability.

### Coverage

The item covers all priming directions from the issue:

**Formal foundations:**
- Catala — prose/spec inversion methodology
- i*/iStar — Eric Yu's original thesis; Strategic Rationale (SR) model for decomposing goals into tasks and resources
- Deon Digital Contract Specification Language (CSL) — normative obligation language separating intent from execution
- Normative consistency and norm conflict detection (AI & Law literature)

**OKR and goal derivability:**
- Hoshin Kanri X-matrix — derivation chains from strategic objectives to initiatives and metrics; machine-checkability question
- Beyond Budgeting (Hope & Fraser) — funding cycle decoupling as structural gap in any strategy formal spec
- Real Options theory (Trigeorgis) — initiative portfolio as option-exercising against stated risk appetite

**The consistency checking problem:**
- Near-decomposability (Simon) applied to intent propagation across abstraction layers
- Rumelt's kernel model (diagnosis, guiding policy, coherent actions) as near-structured specification
- Resource-Based View (RBV) + dynamic capabilities (Teece) — build-vs-buy derivability from formal strategic position

**Also included (as requested):**
- EOSIO Ricardian Template Toolkit — as a concrete working precedent for dual-artefact (machine-readable + human-readable) generation from a single canonical source

### Decisions

- Priority set to **high** — the question is cross-cutting and foundational to several related research threads (goal-oriented requirements, formal contracts, strategy execution)

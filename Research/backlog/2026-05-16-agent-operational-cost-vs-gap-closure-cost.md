# Agent Operational Cost vs Gap Closure Cost

## Research Question

What is the fully loaded operational cost of a production do-mode Artificial Intelligence (AI) agent relative to the cost of closing the underlying systems capability gap via software delivery across three-year and five-year horizons, and under what conditions does do-mode investment produce positive return relative to build-mode investment?

## Scope

**In scope:**
- Cost model components for production do-mode agents: build, inference, harness operations, governance, and failure recovery.
- Cost model components for build-mode closure of integration, functionality, and data access debt classes.
- Breakeven analysis using plausible AI-assisted software engineering productivity multipliers over three and five years.

**Out of scope:**
- Vendor selection for a specific model provider or orchestration platform.
- Organisation-specific budgeting recommendations without comparable benchmark data.

**Constraints:**
- Prioritise empirical cost datasets and benchmark studies over anecdotal case studies.
- Keep assumptions explicit when source data is incomplete.

## Context

The thesis depends on proving that build mode creates higher leverage than do mode over realistic planning horizons. Without a defensible, evidence-backed cost comparison, the do/build decision remains preference-based rather than investment-grade.

## Approach

1. Compile production cost evidence for do-mode agents by complexity and task type, including inference, governance, and recovery overhead.
2. Compile delivery-cost evidence for closing underlying systems capability gaps with AI-assisted software engineering.
3. Model breakeven points and sensitivity to productivity multipliers for integration debt, functionality debt, and data access debt.
